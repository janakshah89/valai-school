General

If you have a question and not answered found here, please contact our support team.

1. Can I Install Fekara on my own hosting?

Yes, feKara could be installed on any Linux based hosting. You would need to buy our One Time Branded Subscription and our technical team would assist you to install the app.

2. Do I need to pay any charges for Renewal?

Yes and also no, In case of monthly subscription user would not pay anything for Support, Hosting, or renewal. But in case of One Time Branded Subscription user would have to pay $100/yr after one of the purchase. The Renewal charges would cover, Support, New modules updates, Cloud Hosting and license renewal.

3. How can I pay for Fekara?

You can pay using a credit card or PayPal for any of the subscriptions for feKara. You would need to place your order on our site and invoice would be sent to you via email, which would be used to pay the pending amount.

4. Does Fekara Support SMS Integration and what do I need for it?

Yes, Fekara School Management System includes the module of SMS. You would need to buy an SMS package from your local vendor. We would need your SMS account to get the API Code and our tech staff would configure your local SMS account with the feKara School Management System.

5. How to run feKara on Local Server Environment?

Although the feKara School Management System is a Web Application it could also be installed on Local server environment but the charges for offline version are entirely different. The user would need a Linux based server environment and would need to have Apache Server installed on the machine and would need MySQL Support.
